# CSS Accordion Cards – City Showcase

An interactive, image-based accordion list built with pure CSS (no JavaScript) using the checked radio-button technique. Clicking a card expands it into a full-height image panel with a gradient overlay and animated text reveal, while collapsing the others — similar to a Netflix-style expanding card UI.

**Key features:**
- Pure CSS interactivity using `:checked` pseudo-class + sibling selectors (`input:checked + label`) — no JavaScript required
- Smooth multi-property transitions (height, width, font-size, opacity, blur) for a polished expand/collapse effect
- Background images with gradient overlays that fade in on selection for text readability
- Responsive layout adjustments for smaller screens
- Clean separation of structure (radio/label pattern) from visual presentation

**Purpose:** Built to deepen CSS skills — particularly state-driven UI without JS, advanced selectors, layered transitions, and background/gradient layering techniques.

**Tech stack:** HTML5, CSS3 (Flexbox, transitions, pseudo-classes, gradients)
